komm.RectangularPulse¶

class RectangularPulse(width=1.0)[source]¶

Rectangular pulse. It is a formatting pulse (FormattingPulse) with impulse response given by

\[\begin{split}h(t) = \begin{cases} 1, & 0 \leq t < w, \\ 0, & \text{otherwise}. \end{cases},\end{split}\]

where \(w\) is the width of the pulse, which must satisfy \(0 \leq w \leq 1\). The rectangular pulse is depicted below for \(w = 1\) (called the NRZ pulse), and for \(w = 0.5\) (called the halfway RZ pulse).
