komm.ManchesterPulse¶

class ManchesterPulse[source]¶

Manchester pulse. It is a formatting pulse (FormattingPulse) with impulse response given by

\[\begin{split}h(t) = \begin{cases} -1, & 0 \leq t < 1/2, \\ 1, & 1/2 \leq t < 1, \\ 0, & \text{otherwise}, \end{cases}\end{split}\]
