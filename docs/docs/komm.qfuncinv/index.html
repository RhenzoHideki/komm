komm.qfuncinv¶

qfuncinv(y)[source]¶

Computes the inverse Gaussian Q-function.

Parameters:

y (float or array of float) – The input to the function. Should be a float or array of floats in the real interval \([0, 1]\).

Returns:

x (same as input) – The value \(x = \mathrm{Q^{-1}}(y)\).

Examples

>>> komm.qfuncinv(0.5)  
0.0

>>> komm.qfuncinv([0.841344746, 0.5, 0.158655254])  
array([-1., 0.,  1.])
