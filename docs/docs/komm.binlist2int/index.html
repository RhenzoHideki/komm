komm.binlist2int¶

binlist2int(binlist)[source]¶

Converts a bit array to its integer representation.

Parameters:

binlist (list or 1D-array of int) – A list or array of \(0\)’s and \(1\)’s whose i-th element stands for the coefficient of \(2^i\) in the binary representation of the output integer.

Returns:

integer (int) – The integer representation of the input bit array.

Examples

>>> komm.binlist2int([0, 1, 0, 1, 1])
26

>>> komm.binlist2int([0, 1, 0, 1, 1, 0, 0, 0])
26
