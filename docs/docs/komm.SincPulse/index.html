komm.SincPulse¶

class SincPulse(length_in_symbols)[source]¶

Sinc pulse. It is a formatting pulse (FormattingPulse) with impulse response given by

\[h(t) = \operatorname{sinc}(t) = \frac{\sin(\pi t)}{\pi t}.\]
