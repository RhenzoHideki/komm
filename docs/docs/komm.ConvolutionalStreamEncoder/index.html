komm.ConvolutionalStreamEncoder¶

class ConvolutionalStreamEncoder(convolutional_code, initial_state=0)[source]¶

Convolutional stream encoder. Encode a bit stream using a given convolutional code (ConvolutionalCode). The internal state of the encoder is maintained across each call.

Examples

>>> convolutional_code = komm.ConvolutionalCode([[0o7, 0o5]])
>>> convolutional_encoder = komm.ConvolutionalStreamEncoder(convolutional_code)
>>> convolutional_encoder([1, 1, 1, 1])
array([1, 1, 0, 1, 1, 0, 1, 0])
>>> convolutional_encoder([1, 1, 1, 1])
array([1, 0, 1, 0, 1, 0, 1, 0])
