komm.RaisedCosinePulse¶

class RaisedCosinePulse(rolloff, length_in_symbols)[source]¶

Raised cosine pulse. It is a formatting pulse (FormattingPulse) with impulse response given by

\[h(t) = \operatorname{sinc}(t) \frac{\cos(\pi \alpha t)}{1 - (2 \alpha t)^2},\]

where \(\alpha\) is the roll-off factor. The raised cosine pulse is depicted below for \(\alpha = 0.25\), and for \(\alpha = 0.75\).

   

For \(\alpha = 0\), the raised cosine pulse reduces to the sinc pulse (SincPulse).
